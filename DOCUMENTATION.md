# Framer UI Vault - User Documentation

## What is Framer UI Vault?

Framer UI Vault is a premium plugin for Framer that provides a library of pre-built, fully editable UI components. Instead of building layouts from scratch, simply browse, preview, and insert professional components directly into your Framer canvas.

---

## Getting Started

### Installation
1. Open your Framer project
2. Go to **Plugins** in the toolbar
3. Search for "Framer UI Vault" or install from the plugin marketplace
4. Click **Install**

### Opening the Plugin
- Click the **Framer UI Vault** icon in your plugins panel
- Or use the keyboard shortcut (if configured)

---

Customize components before inserting:

- **Search** - Type to find components by name

### 4. Favorites & Recently Used
- **Favorites** - Click the heart icon to save frequently used components
- **Recently Used** - Quick access to your last 10 inserted components

## How to Use

### Inserting a Component

1. **Browse** - Scroll through components or use search/filters
2. **Preview** - Hover over a component for 500ms to see enlarged preview
3. **Insert** - Click the component card to insert into your canvas

The component will be inserted as native Framer frames, fully editable!

### Customizing Theme Colors

1. Click the **palette icon** (🎨) in the header
2. Choose **Light** or **Dark** mode
3. Select an accent color from presets or use the color picker
4. All previews update instantly
5. Insert components - they'll use your selected theme

### Using Favorites

1. Hover over any component card
2. Click the **heart icon** (♡) to add to favorites
3. Click **Favorites** in the sidebar to view saved components
4. Click the heart again to remove from favorites

### Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl/Cmd + F` | Focus search input |
| `Escape` | Clear search / Close panels |
| `Enter` | Insert focused component |
| `Arrow Keys` | Navigate component grid |
| `Tab` | Navigate between elements |

## Tips & Best Practices

### For Best Results
- **Select a frame first** - Components insert better when you have a target frame selected
- **Use theme customization** - Match your brand colors before inserting
- **Check the preview** - Hover to see the full component before inserting

### After Inserting
- All components are **native Framer frames** - fully editable
- Change text, colors, spacing, and layout as needed
- Components are not linked - each insert is independent

### Performance
- The plugin uses virtualization for smooth scrolling with large libraries
- Components are lazy-loaded for fast initial load
- Parsed SVGs are cached for quick re-insertion

## Troubleshooting

### Component doesn't insert
- Make sure you have canvas permissions enabled
- Try selecting a frame on the canvas first
- Check the toast notification for error details

### Preview not showing
- Hover for at least 500ms to trigger preview
- Move mouse away and hover again

### Search not finding components
- Check spelling
- Try partial words (e.g., "head" instead of "header")
- Clear search and browse categories instead

### Theme colors not applying
- Ensure you've selected a theme mode (light/dark)
- Custom colors must be valid hex format (#RRGGBB)
- Click "Reset" to restore defaults
