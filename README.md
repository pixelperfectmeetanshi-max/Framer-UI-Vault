# Framer UI Vault Plugin

Framer UI Vault is a powerful and lightweight Framer plugin designed to supercharge your design workflow. It instantly provisions beautiful, meticulously structured, and fully responsive **Skeletal UI Layouts** directly into your Framer canvas.

## Key Features

- **Instant High-Fidelity Skeletons**: Stop wiring up rectangles from scratch. Instantly drag-and-drop or click to insert massive Hero sections, Bento box grids, Ecommerce product galleries, and more!
- **100% Native Architecture**: The plugin uses a proprietary extraction engine ensuring that components naturally map themselves to Framer's absolute positioning or canvas viewports, avoiding nested coordinate crashes.
- **Copyright Safe**: All internal layouts focus purely on primitive geometry (`<rect>` and `<circle>`). It contains zero external Unsplash resources or arbitrary fonts that could violate guidelines or cause missing asset bugs in Framer.
- **Zero Configuration**: No API keys, no logins required. It works the moment you open it.
- **Light & Dark Mode**: A beautiful, clean developer UI utilizing your preference from OS settings or manual toggle.

## How to Use

1. Launch **Framer UI Vault** from the Plugins menu.
2. Select your category from the left Sidebar.
3. Click "Insert" on any component card.
4. The requested layout will be magically painted onto your explicit canvas context as a flawlessly scalable hierarchy of Native SVGs and Layout Frames.
5. Resize, override colors, and replace the placeholder rectangles with your own real text strings and image links!
